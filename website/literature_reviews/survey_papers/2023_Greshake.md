
# 2023 Greshake 


## Title: Not what you’ve signed up for: Compromising Real-World LLM-Integrated Applications with Indirect Prompt Injection



## Summary

- This paper introduces Indirect Prompt Injection attacks, where adversaries embed prompts in external data to remotely manipulate the behavior of LLM-integrated applications, proposes a security taxonomy of the attack vectors and impacts (e.g., data exfiltration, API manipulation, information ecosystem contamination), and demonstrates their practical feasibility on real-world systems.

- 이 논문은 LLM-기반 애플리케이션에서 외부 데이터에 삽입된 프롬프트를 통해 모델의 동작을 원격으로 조작하는 Indirect Prompt Injection 공격을 제안하고, 데이터 탈취·API 조작·정보 생태계 오염 등 다양한 보안 영향을 포함한 공격 taxonomy를 제시하며 실제 시스템에서 그 실현 가능성을 실험으로 검증한 연구이다.
